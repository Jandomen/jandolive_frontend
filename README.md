Jandolive

Jandolive es una aplicación de chat en tiempo real que permite a los usuarios conectarse mediante video y audio, así como enviar mensajes de texto. Ideal para conversaciones anónimas y seguras con personas de todo el mundo.

Características

Video y audio en tiempo real usando WebRTC.

Chat de texto integrado con mensajes en tiempo real.

Diseño responsivo y moderno con Tailwind CSS.

Arrastrar y soltar ventanas de video y chat para personalizar la vista.

Indicadores de estado de cámara y micrófono.

Mensajes únicos sin duplicados y con formato profesional.

Conexión instantánea mediante sockets con Socket.IO.
